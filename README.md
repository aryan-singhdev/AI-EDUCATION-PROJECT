# AI-EDUCATION-PROJECT

## Edu-AI An Powered Personalized Learning Platform

A modern AI-based education platform designed to provide personalized learning support and improve the learning experience using web technologies.

## Overview

This project focuses on creating a smart learning environment where users can access educational resources with an improved and interactive interface.

## Features

- AI-powered learning assistance
- Personalized learning experience
- Responsive design for mobile and desktop
- Interactive user interface
- API integration
- Modern web-based learning platform

## Tech Stack

- HTML
- CSS
- JavaScript
- React.js
- API Integration
- Vercel Deployment
